Array-polyfill

Array polyfill of common methods for (older) browsers (especially for IE7 and 8)

View project on GitHub

Polyfills for common Array methods

Use them as a polyfill, especially for Internet Explorer 7 and 8.

Based on the great reference at MDN Arrays.

Currently available as single files or a combined file:

  • Array.every
  • Array.some
  • Array.filter
  • Array.forEach
  • Array.indexOf
  • Array.reduce

Installation

Clone github repository: git clone https://github.com/zhongdeliu/array-polyfill.git

or via Bower: bower install array-polyfill (if needed, use --save-dev parameter)

Then include per script tag, before your scripts:

<script type="text/javascript" src="[path-to-file]/array-polyfill.min.js"></script>
